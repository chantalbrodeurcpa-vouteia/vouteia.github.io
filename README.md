> L'IA qui tourne sur vos propres données d'entreprise — et nulle part ailleurs.

---

## Ce que nous construisons

Voûte IA est une plateforme d'intelligence artificielle déployée entièrement sur l'infrastructure du client. Aucune donnée ne transite vers un serveur externe. Aucune connexion internet requise. Le modèle est entraîné sur les données réelles de l'organisation et devient un expert de son secteur.

Nous ciblons les PME, les entreprises industrielles et les institutions qui ont besoin de la puissance de l'IA sans sacrifier la confidentialité de leurs données d'affaires.

---

## Pourquoi l'IA privée

Les outils IA grand public (ChatGPT, Copilot, Gemini) envoient vos données vers des serveurs étrangers. Pour une PME dont les données financières, opérationnelles ou clients sont sensibles, cette réalité pose trois problèmes concrets :

- **Loi 25 (Québec)** — Tout transfert de renseignements personnels hors Québec requiert une Évaluation des Facteurs relatifs à la Vie Privée (EFVP). Sans elle, l'organisation s'expose à des sanctions significatives.
- **Risque concurrentiel** — Vos stratégies, vos marges, vos données clients peuvent alimenter les modèles d'apprentissage de tiers.
- **Dépendance externe** — Un changement de tarification ou une interruption de service d'un fournisseur IA peut paralyser vos opérations.

Voûte IA résout ces trois problèmes à la racine, par l'architecture.

## Applications

```
Audit de processus financiers        →  Cartographie, goulots, chiffrage des gains
Assistant de clôture comptable       →  Checklist guidée, détection d'anomalies, commentaires de gestion
Analyse de données opérationnelles   →  Rapports, tendances, alertes sur écarts
Conformité réglementaire             →  Documentation automatisée, préparation aux audits
Aide à la décision                   →  Synthèse de données internes pour les dirigeants
```

---

## Architecture technique

```
Modèle de base          →  Open source (Mistral, LLaMA) — aucune dépendance propriétaire
Fine-tuning             →  Entraînement sur les données sectorielles du client
Déploiement             →  Infrastructure du client ou serveur dédié canadien
Connectivité            →  100% hors ligne possible — zéro appel externe
Conformité              →  Conforme Loi 25 par conception, auditable en tout temps
```
